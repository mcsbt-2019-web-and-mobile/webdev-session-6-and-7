lifecycle

React is a very fast framework because it makes a good use of resources. - renders components only when they are in the screen - destroys components that are not seen - renders only the needed part of the component

There main method we want to pay attention to is componentDidMount().

what’s mounting?

Mounting is the term used in React to put elements in the DOM, unmount is the opposite, the action of removing them from the DOM.

componentDidMount()

componentDidMount() is the method executed just after the component is rendered. We should use this method to put all side effects that would update the state of the component.

  • call an API
  • modify the DOM …
